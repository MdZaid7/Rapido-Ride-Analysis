# Rapido-Ride-Analysis
# 🚖 Rapido Ride Analysis Dashboard

This is an end-to-end Data Analytics project built using **SQL** and **Power BI**.

In this project, I analyzed **20,000+ Rapido ride bookings** to understand booking trends, cancellations, vehicle performance, location insights, and customer behavior. The main goal was to convert raw ride booking data into meaningful business insights through interactive dashboards.

---

## 🛠 Tools Used

- SQL
- Microsoft Power BI
- Power Query
- DAX

---

## 📊 Dashboard Highlights

- Total Bookings
- Total Booking Value
- Average Ride Distance
- Driver & Customer Ratings
- Booking Trend by Day
- Payment Method Analysis
- Vehicle-wise Performance
- Top Pickup & Drop Locations
- Driver Cancellation Hotspots
- Time Slot Analysis
- Booking Heatmap (Hour × Day)

---

## 📂 Project Workflow

### SQL

- Cleaned the raw dataset
- Removed unnecessary data
- Created Time Slot
- Created Weekday/Weekend column
- Wrote business queries for analysis

### Power BI

- Built data model
- Created DAX measures
- Designed interactive dashboard
- Added slicers and dynamic filtering
- Built KPIs and business insights

---

## 📈 Key Insights

- Identified the busiest pickup locations.
- Found areas with the highest driver cancellations.
- Compared booking patterns across weekdays and weekends.
- Analyzed vehicle preference by customers.
- Estimated pickup time using Driver TAT since the dataset didn't contain actual pickup timestamps.
